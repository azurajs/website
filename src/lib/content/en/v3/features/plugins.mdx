# Plugins

AzuraJS ships with **15 built-in plugins** that cover security, performance, real-time communication, infrastructure, and more. Every plugin is a zero-dependency module built on Node.js primitives, fully typed, and designed to compose cleanly with one another.

## How Plugins Work

A plugin is a function that receives a `PluginContext` (containing `req` and `res`) and a `next` callback. Plugins run **before** global middleware and route handlers.

```typescript
import { AzuraServer } from "azurats";
import { CORSPlugin } from "azurats/plugins";

const app = new AzuraServer();

app.plugin(CORSPlugin({ origins: ["https://example.com"] }));

app.listen(3000);
```

Register a plugin with `app.plugin()`. Each factory function accepts an options object and returns a `PluginHandler`.

## Plugin Execution Order

Plugins execute in registration order. A recommended production ordering:

```typescript
const app = new AzuraServer();

// 1. Security
app.plugin(HelmetPlugin());
app.plugin(CORSPlugin({ origins: [process.env.FRONTEND_URL!] }));

// 2. Observability
app.plugin(RequestIdPlugin());

// 3. Performance
app.plugin(CompressionPlugin({ threshold: 1024 }));
app.plugin(TimeoutPlugin({ timeout: 30_000 }));

// 4. Rate limiting
app.plugin(RateLimitPlugin({ windowMs: 60_000, max: 100 }));

// 5. Authentication
app.plugin(JWTPlugin({ secret: process.env.JWT_SECRET! }));

// 6. Monitoring
app.plugin(HealthCheckPlugin({ path: "/health" }));
```

<Callout>
Plugins registered with `app.plugin()` always run before middleware added with `app.use()`. Within each group, execution follows registration order.
</Callout>

## Creating a Custom Plugin

Any function matching the `PluginHandler` signature works as a plugin:

```typescript
import type { PluginContext, NextFunction } from "azurats/plugins";

function TimingPlugin() {
  return (ctx: PluginContext, next: NextFunction) => {
    const start = process.hrtime.bigint();

    const originalEnd = ctx.res.end.bind(ctx.res);
    ctx.res.end = (...args: any[]) => {
      const ms = Number(process.hrtime.bigint() - start) / 1e6;
      ctx.res.header("X-Timing", `${ms.toFixed(2)}ms`);
      return originalEnd(...args);
    };

    next();
  };
}

app.plugin(TimingPlugin());
```
